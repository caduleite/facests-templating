Enquanto densenvolvendores dentro de uma IDE (ambiente integrado de desenvolvimento) a primeira coisa que devemos fazer é criar um novo projeto para armazenar os nossos arquivos, arquivos fontes, arquivos de configuração e demais recursos para a aplicação no caso do desenvolvimento Web Java isso nao é diferente.

JFS seria um framework ou seja biblioteca para desenvolver software, ou seja é um abstração que reuni códigos em comum que podem ser utilizado em diferentes apliçaçoes somente chamado seus objetos. Permitindo a separação entre lógica e regras de negocio, navegação, conexao com servicos externos e gereciamento de configurações e de um modelo de interfaces graficas baseado em eventos.
